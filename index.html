Master's Thesis

My master's thesis was developed in High Performance Computing (HPC) field, where it was supervised by Prof. Alberto José Proença (Adviser) and Artur Mariano (Co-adviser). The thesis is entitled 'Towards an efficient lattice basis reduction implementation'.

The security of most digital systems is under serious threats due to major technology breakthroughs we are experienced in nowadays. Lattice-based cryptosystems are one of the most promising post-quantum types of cryptography, since it is believed to be secure against quantum computer attacks. Their security is based on the hardness of the Shortest Vector Problem and Closest Vector Problem.

Integrit

Integrit is a Java Service that implements a File Verification System on Unix.

Integrit

The project aims to implement a background Java Service that is started automatically when the system is turned on. It is a File Verification System on Unix. The service is also able to recover its previous state, by saving its state into a file. It can be started, paused, and stopped at any time with the use of system signals, i.e., HUP and INT.

Every time that someone access a file or directory, it is left a trace. Thus, it is possible to prevent/avoid possible attacks to sensitive information. This project verifies:

  • Last access to a directory/file;
  • Last time a file was edited;
  • New directories/files;
  • Deleted directories/files;
  • Ownership's changes;
  • Group's changes;
  • Change of access permissions;

Then, there are done system logs with different levels of danger.

Secure Chat-Server

Secure Chat-Server is a simple chat-room that implements the best cryptographic tecniques.

Secure Chat-Server

Secure Chat-Server is a simple chat-room that implements the best cryptographic techniques that should be implemented in secure everyday applications. This project is constituted by two main applications (Client-side and Server-side).

The messages are encrypted and decrypted by the chosen cryptosystem, i.e., CBC. At the beginning of each communication, the key is agreed by the two entities with the Diffie–Hellman key exchange protocol.

Unfortunately, these techniques cannot avoid the well-known man-in-the-middle attack. Thus, it implements digital signatures, where we simulate a Certification Authority (CA) to generate and sign the digital certificates. These digital certificates are managed with X.509 standard for a Public Key Infrastructure (PKI).

3D Graphics Engine

3D Graphics Engine is a C++ engine that is capable of rendering 2D and 3D graphics.

3D Graphics Engine

3D Graphics Engine is a C++ application that uses the OpenGL library. It was developed within the class of Computer Graphics. The project requires two main applications: (1) a generator that generates all the necessary points to render an arbitrary 3D object, i.e., sphere, cube, and (2) a graphics engine that renders the 3D objects from different sources.

These project aims to consolidate the knowledge learned during the classes. The generator is responsible for modelling objects, using surfaces defined analytically, where it generates a GLUT version and a Vertex Buffer Objects (VBO's) version to achieve better performance. In addition, it also generates cubic surfaces, such as Bézier surfaces.

However, the most important techniques are implemented in the graphics engine, which implements geometric transformations, culling, Catmull-Rom curves, illumination effects, textures, atlas textures, pop-up menus, picking, different camera modes (explorer mode / first-person), View Frustum Culling.

The funny part of this project is the graphics engine, that loads all the information to render a scene from a XML file, where each object can have different properties without affect the others.

DBLP

DBLP is a C language application for statistical processing of bibliographic information.

DBLP

The DBLP Computer Science Bibliography (http://dblp.org/db/) is a website for non-commercial purposes to store and view bibliographic information on scientific articles (in journals and conferences) in IT. It is intended to develop an application in C language that allows to statistical processing of bibliographic information obtained in DBLP for articles published in a number of magazines and/or conferences.

One of the goals of this project is to exercise best practices and techniques acquired in a practical case study of medium size, where the construction of the project follows some important principles, such as:

  • The choice of data structures for representation and processing of information that addresses the problem, i.e., Hash tables, AVL trees and linked lists;
  • The use of algorithms and settings showed in Imperative Programming and Algorithms and Complexity, i.e., binary searches;
  • Writing generic, modular, reusable code, which can be utilized in other future projects;
  • The analysis and optimization of application performance, using the matter addressed in Computer Architecture.

Illuminatus

Illuminatus is a puzzle game that aims to light up all unlocked cells in the board.

Illuminatus

The project aims to implement an application to solve a kind of puzzle called Illuminatus. The objective of this puzzle is to light up all the houses unlocked of a checkerboard. A house is light up if:

  • Have a lamp;
  • Have a lamp in the same row or column and n to be no locked house between the house and that continued in the lamp.

The application was developed in the C programming language on Linux operating system, where it was necessary to study several strategies to solve the puzzle.
